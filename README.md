# Civilization VII Mod Manager

Mod management tool for Civilization VII that allows you to download, install, and automatically update mods from Civfanatics.

Currently supported on Windows and Apple Silicon Mac.

## Installation
You can download the latest release here:  
[Download Civ7 Mod Manager](https://github.com/haruleekim/civ7-mod-manager/releases/latest)
